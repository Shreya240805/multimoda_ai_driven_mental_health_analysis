# multimoda_ai_driven_mental_health_analysis
Multimodal Mental Health Analysis System fuses facial, vocal, and cardiovascular data using AI (OpenCV, DLib, DeepFace, Librosa). Correlates with PHQ-9 for clinical relevance. Built in Python with Flask, TensorFlow, SQLAlchemy. Demonstrates interdisciplinary AI, signal processing, healthcare skills. Highlights research acumen and real-world impact.
